## 技术架构

### ECS 组件设计

```
组件层 (Components):
├── Position - 棋盘位置
├── Stone - 棋子属性
├── GameBoard - 棋盘状态
├── GameState - 游戏状态
├── GameStats - 统计数据
├── AIPlayer - AI配置
├── UIState - 界面状态
└── MouseState - 鼠标交互
```

### 系统架构

```
系统层 (Systems):
├── InputSystem - 输入处理
├── BoardSystem - 棋盘逻辑
├── GameLogicSystem - 游戏规则
├── AISystem - AI决策
├── RenderSystem - 图形渲染
└── UISystem - 界面管理
```

### 场景管理

```
场景层 (Scenes):
├── MenuScene - 主菜单
└── GameScene - 游戏主界面
```

## 运行方式

### 🚀 快速启动

```bash
# 方式1: 直接运行
uv run go_game/main.py

# 方式2: 使用启动器
uv run go_game/launcher.py

# 方式3: Web版本
python -m pygbag go_game/pygbag_launcher.py
```

### 🧪 功能测试

```bash
uv run go_game/test.py
```

## 游戏操作

| 操作     | 功能                 |
| -------- | -------------------- |
| 鼠标点击 | 在棋盘交叉点放置棋子 |
| P 键     | 跳过当前回合         |
| R 键     | 重新开始游戏         |
| ESC 键   | 返回主菜单           |
| ↑/↓ 键   | 菜单导航             |
| 回车键   | 确认选择             |
