# Multi-Agent System

## Game abstract

目前有两个势力，红方和白方，每个势力有四个角色类型，分别是主公、山、水和平。
主公是势力的核心，有且仅有一个，山、水、平是势力的维护者，可以有多个。每个类型都有自己的属性，如擅长水战、擅长山战等。
（串行）
每个角色都有自己的行动回合，每个回合可以选择移动、攻击、使用技能等。游戏的目标是消灭对方的主公。
（并行）
所有角色共享一个时间回合，同时进行行动，每个角色的行动都会影响到其他角色的行动。

## Game Analysis

假设任一可以移动单位是一个Agent，可以进行观察，思考，决策，执行。

观察，每个agent 可以观察自身周围的环境，以及本势力其他agent的位置和环境等。

可以观察到 全局地图 以及其自身或队友周围有没有其他势力的agent及类型。

思考，每个agent可以根据自己的属性，周围环境，以及其他势力的agent的位置等，进行思考，决策。

决策，每个agent可以根据思考的结果，决定自己的行动，如移动，攻击等。

执行，每个agent可以根据决策的结果，执行自己的行动。
