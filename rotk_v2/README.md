# 三国演义游戏 (Romance of the Three Kingdoms)

## 项目概述

这是一个基于自定义游戏引擎框架（framework_v2）开发的即时策略游戏。游戏以三国演义为背景，玩家需要在复杂的局势中做出决策，通过合理的策略击败对手并获取胜利。游戏通过场景管理系统实现不同游戏阶段的切换，包括开始场景、游戏主场景和结束场景。

## 游戏设计目标

考察玩家的推理能力和决策能力，让玩家在游戏中体验到不同环境下，因为约束条件不同，对任务分析处理方式的不同，以及对任务处理结果的不同。

## 游戏特性

- 基于ECS（实体-组件-系统）架构设计
- 真实尺寸的地图系统（10km x 10km）
- 多样化的地形元素（平原、高原、丘陵、山地、盆地、河流、森林、湖泊等）
- 多阵营对抗系统
- 多种兵种类型（步兵、骑兵、弓兵、弓骑兵、飞行单位）
- 任务系统（移动、攻击、防御、巡逻）
- 实时战略游戏玩法
- 简洁的图形用户界面

## 游戏玩法

1. 玩家控制蜀国阵营，与魏国和吴国进行对抗
2. 通过选择单位并发布命令来控制战场
3. 利用不同地形的特性制定战略
4. 合理分配资源和兵力
5. 消灭所有敌对阵营以获取胜利

## 如何运行

在项目根目录下执行以下命令：

```bash
python rotk_v2/game.py
```

## 操作说明
- 鼠标左键：选择单位或框选多个单位
- 鼠标右键：移动选中的单位或攻击敌方单位
- WASD或方向键：移动相机
- 鼠标滚轮：缩放视图
- ESC键：打开游戏菜单
