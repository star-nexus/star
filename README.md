# Romance-of-the-Three-Kingdoms
An LLM Multi-agent Environment in Romance of the Three Kingdoms 

## Get Started

### Install 

```bash
pip install -r requirements.txt
```

### Run
```bash
python game.py
```

## Game Rules   

### Battle
游戏单位克制关系
```
山 克 平
平 克 水
水 克 山
```

### Control
鼠标选择游戏单位，使用上下左右进行移动

键盘 1、2、3选择视角
```
1:上帝视角
2:红方视角
3:白方视角
```
