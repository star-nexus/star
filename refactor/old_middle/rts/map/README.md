# 地图模块 (Map)

地图模块负责RTS游戏中的地图创建、渲染和路径查找功能，为游戏提供地形和导航系统。

## 模块概述

地图系统是RTS游戏的基础，它定义了游戏世界的物理环境，包括不同类型的地形、障碍物以及单位移动的基本规则。

## 主要组件

### 地形类型 (TileType)

定义地图上的地形类型常量和属性：
- 平原、山地、水面、森林、沼泽等地形类型
- 地形颜色映射（用于渲染）
- 地形通行难度系数
- 地形防御加成

### 地图格子 (Tile)

表示地图上的一个格子单元：
- 坐标位置（x，y）
- 地形类型
- 通行性
- 占据此格子的实体
- 地形属性（如通行难度、防御加成）

### 地图数据 (MapData)

存储和管理地图数据：
- 地图尺寸（宽度和高度）
- 格子字典，以(x,y)为键
- 格子访问和管理方法
- 邻居和范围查询功能

### 地图生成器 (MapGenerator)

负责生成随机地图：
- 创建不同复杂度的随机地图
- 生成各种地形特征（水域、山脉、森林、沼泽）
- 从文件加载地图（待实现）

### 地图渲染器 (MapRenderer)

负责将地图数据渲染到屏幕：
- 地形纹理加载和管理
- 视口移动和缩放
- 坐标转换（屏幕坐标<->地图坐标）
- 可见区域优化渲染

### 寻路系统 (PathFinder)

实现A*寻路算法：
- 在地图上查找从起点到终点的路径
- 处理不同地形的移动代价
- 支持特殊移动能力（穿越水面、山地）
- 寻找最近的可通行点

## 地形特征

地图支持多种地形类型，每种类型对单位有不同影响：

1. **平原 (Plains)**：
   - 基本地形，标准移动速度
   - 无防御加成

2. **山地 (Mountain)**：
   - 移动困难，速度减半
   - 提供30%防御加成
   - 特殊单位可穿越

3. **水面 (Water)**：
   - 默认不可通行
   - 特殊单位可以通行但速度慢
   - 水中减少10%防御

4. **森林 (Forest)**：
   - 移动稍慢
   - 提供20%防御加成（隐蔽）

5. **沼泽 (Swamp)**：
   - 移动非常困难
   - 提供10%防御加成

## 使用示例

```python
# 创建地图生成器和地图数据
map_generator = MapGenerator()
map_data = map_generator.generate_random_map(width=50, height=50, complexity=0.7)

# 创建地图渲染器
map_renderer = MapRenderer(tile_size=32)

# 初始化寻路系统
pathfinder = PathFinder(map_data)

# 查找路径
path = pathfinder.find_path(start_x=5, start_y=5, goal_x=25, goal_y=25)

# 渲染地图
map_renderer.render(screen, map_data)

# 转换坐标
map_x, map_y = map_renderer.screen_to_map(screen_x=400, screen_y=300)
```

## 地图生成流程

地图生成过程中按以下顺序创建地形特征：

1. 首先生成基本的平原地形作为默认
2. 添加水域（湖泊和河流）
3. 生成山脉和山地
4. 添加森林区域
5. 创建沼泽（优先在水域附近）
