## 🔧 实现的功能

### 1. 实时模式 (Real-time Mode)

#### 核心改动：
- **TurnSystem** 增加实时模式支持
  - 添加了`_update_real_time()`方法处理实时模式逻辑
  - 实现了`_regenerate_action_points()`实时恢复单位行动力
  - 添加了`_update_ai_decisions()`实时AI决策更新
  - 修改了构造函数，支持从外部传入游戏模式

#### 游戏机制：
- 实时模式下单位行动力自动恢复（每秒恢复50%）
- AI可以实时做决策而不需要等待回合
- 战斗冷却时间机制
- 保留了游戏结束检查

#### 启动方式：
```bash
python rotk/main.py --mode real_time     # 实时模式
python rotk/main.py --mode turn_based    # 回合制模式（默认）
```

### 2. 小地图 (MiniMap)

#### 组件设计：
- **MiniMap组件** (`rotk/components/minimap.py`)
  - 可配置的显示位置和大小
  - 支持显示/隐藏地形、单位、摄像机视口
  - 可点击交互导航
  - 可自定义样式（颜色、透明度、边框等）

#### 渲染功能：
- **地形渲染**: 不同地形类型用不同颜色表示
- **单位渲染**: 根据阵营显示不同颜色的圆点
- **摄像机视口**: 白色矩形框显示当前视野范围
- **边框装饰**: 可配置的边框样式

#### 交互功能：
- **点击导航**: 点击小地图任意位置，摄像机会移动到对应区域
- **坐标转换**: 屏幕坐标 ↔ 小地图坐标 ↔ 世界坐标的精确转换
