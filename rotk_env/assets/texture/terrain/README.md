# 地形贴图系统

## 目录结构

地形贴图按照地形类型分类存放在对应的子目录中：

```
terrain/
├── plain/          # 平原贴图
├── forest/         # 森林贴图
├── mountain/       # 山地贴图
├── hill/           # 丘陵贴图
├── water/          # 水域贴图
└── city/           # 城池贴图
```

## 支持的图片格式

- PNG (.png)
- JPEG (.jpg, .jpeg)

## 贴图选择

- 每个地形类型目录可以包含多个贴图文件
- 系统会为每个地图格子随机选择一个贴图（基于格子坐标的固定种子）
- 选择的贴图会被缓存，确保同一个格子始终显示相同的贴图

## 贴图规范

- 建议贴图尺寸为正方形（如 64x64, 128x128 等）
- 系统会自动将贴图缩放到合适的大小
- 贴图文件名可以任意命名

## 后备方案

如果某个地形类型没有贴图文件，系统会自动回退到使用纯色渲染。
