# 基础 ECS 示例

这个示例展示了 ECS 架构的基本概念和用法。

## 功能特点

- **组件系统**: 展示如何定义和使用组件（NameComponent, HealthComponent, LevelComponent）
- **系统架构**: 实现了三个不同功能的系统
- **实体管理**: 演示实体的创建和组件的添加
- **数据查询**: 展示如何查询具有特定组件的实体

## 包含的系统

1. **StatusDisplaySystem** - 状态显示系统

   - 定期显示所有角色的详细状态
   - 包括姓名、等级、生命值、经验等信息

2. **ExperienceSystem** - 经验系统

   - 自动给角色增加经验
   - 处理角色升级逻辑
   - 升级时恢复生命值

3. **CombatSimulationSystem** - 战斗模拟系统
   - 模拟战斗中的伤害
   - 处理角色死亡

## 运行示例

```bash
cd examples/basic_ecs
python main.py
```

## 预期输出

程序会创建几个三国英雄角色，然后你可以观察到：

- 角色状态的实时显示
- 经验值的增长和升级
- 战斗伤害和死亡
- ECS 系统间的协调工作

## 学习要点

1. **组件设计**: 组件只包含数据，不包含逻辑
2. **系统职责**: 每个系统专注于特定的功能
3. **实体查询**: 通过组件类型查询相关实体
4. **松耦合**: 系统间通过组件数据交互，而非直接调用
