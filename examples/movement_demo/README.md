# 移动系统示例

这个示例展示了如何在 ECS 架构中实现 2D 空间的移动系统。

## 功能特点

- **基础移动**: 位置和速度组件的基本移动
- **边界碰撞**: 实体碰到边界时的反弹效果
- **AI 寻路**: 自动寻找目标并移动
- **多种移动模式**: 不同类型的实体有不同的移动行为

## 包含的组件

1. **PositionComponent** - 存储 2D 位置坐标
2. **VelocityComponent** - 存储速度矢量和最大速度限制
3. **TargetComponent** - 存储 AI 移动的目标位置
4. **BoundsComponent** - 定义移动边界
5. **NameComponent** - 实体名称

## 包含的系统

1. **MovementSystem** - 基础移动系统

   - 根据速度更新位置
   - 处理基本的物理移动

2. **BoundarySystem** - 边界系统

   - 检测边界碰撞
   - 实现反弹效果

3. **AIMovementSystem** - AI 移动系统

   - 自动寻路到目标位置
   - 到达目标后设置新的随机目标
   - 计算方向向量和速度

4. **StatusSystem** - 状态显示系统
   - 显示实体的位置、速度、目标等信息

## 运行示例

```bash
cd examples/movement_demo
python main.py
```

## 预期输出

程序会创建几个不同类型的移动实体：

- **游侠、骑兵、弓箭手、法师**: 有 AI 的实体，会自动寻找目标
- **弹跳球**: 纯物理移动，在边界内反弹

观察现象：

- 实体在 2D 空间中移动
- AI 实体会智能寻路到目标点
- 碰到边界时会反弹
- 不同实体有不同的移动速度

## 学习要点

1. **组件组合**: 不同组件组合产生不同行为
2. **系统协作**: 多个系统协同工作实现复杂功能
3. **物理模拟**: 基本的 2D 物理移动和碰撞
4. **AI 行为**: 简单的寻路和目标选择算法
5. **状态管理**: 实时监控和显示实体状态
