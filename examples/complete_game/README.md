# 完整游戏示例 - 三国策略小游戏

这是一个使用完整 framework 的策略小游戏示例，展示了 ECS 架构在实际游戏中的应用。

## 游戏特点

- **实时策略**: 多个阵营的单位在 2D 空间中移动和交互
- **AI 行为**: 不同的 AI 行为模式（漫游、追逐）
- **用户交互**: 鼠标点击选择单位
- **图形界面**: 使用 Pygame 进行渲染
- **完整架构**: 使用 framework 的引擎、场景、事件系统

## 游戏内容

### 三个阵营

1. **刘备军（蓝色）**: 玩家阵营，漫游行为
2. **曹操军（红色）**: 敌对阵营，会追逐其他阵营
3. **孙权军（绿色）**: 中立阵营，漫游行为

### 游戏机制

- **移动系统**: 单位在 2D 空间中移动
- **AI 系统**: 不同的 AI 行为模式
- **选择系统**: 鼠标点击选择单位
- **渲染系统**: 实时图形渲染
- **UI 系统**: 游戏信息和选中单位的详情显示

## 系统架构

### 组件 (Components)

- `PositionComponent` - 2D 位置
- `RenderComponent` - 渲染属性（颜色、大小、形状）
- `NameComponent` - 单位名称
- `HealthComponent` - 生命值
- `VelocityComponent` - 移动速度
- `AIComponent` - AI 行为配置
- `FactionComponent` - 阵营归属
- `SelectableComponent` - 选择状态

### 系统 (Systems)

- `RenderSystem` - 处理图形渲染
- `MovementSystem` - 处理移动逻辑
- `AISystem` - 处理 AI 行为
- `SelectionSystem` - 处理用户选择
- `UISystem` - 处理用户界面

## 运行要求

确保安装了 pygame：

```bash
pip install pygame
```

## 运行示例

```bash
cd examples/complete_game
python main.py
```

## 控制方式

- **左键点击**: 选择单位
- **关闭窗口**: 退出游戏

## 观察要点

1. **ECS 架构**: 观察组件和系统的协作
2. **AI 行为**: 红色单位会追逐其他阵营
3. **用户交互**: 点击选择单位查看详情
4. **实时渲染**: 流畅的图形更新
5. **系统解耦**: 各系统独立工作但协调配合
