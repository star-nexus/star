# 工厂模块 (Factories)

工厂模块负责创建和配置游戏中的各类实体，采用工厂模式来封装实体创建的复杂过程。

## 模块概述

工厂负责创建游戏中的各类实体，包括单位、建筑和资源节点。每个工厂专注于一类实体的创建，提供简洁的接口来生成完全配置的实体。

## 主要工厂

### 单位工厂 (UnitFactory)

负责创建各种类型的单位实体：
- 创建不同类型的单位（辎重、平原、山地、水面、远程、空中）
- 配置单位的组件和属性
- 生成临时的单位视觉表示（用于开发阶段）

支持的单位类型：
- 辎重单位：高资源携带量，低战斗力，低速度
- 平原单位：标准属性，平原上速度加快
- 山地单位：高攻击力，可穿越山地，山地上速度快
- 水面单位：可在水上移动，水上速度快
- 远程单位：攻击范围远，低防御，低速度
- 空中单位：忽略地形惩罚，高速度，相对脆弱

### 建筑工厂 (BuildingFactory)

负责创建各种建筑实体：
- 创建不同类型的建筑（主基地、补给站、防御工事）
- 配置建筑的组件和属性
- 管理建筑的视觉表示和占地面积

支持的建筑类型：
- 主基地：生产各种资源，可以生产单位
- 补给站：生产食物，提供资源存储
- 防御工事：提供额外的防御力，抵抗不同类型的攻击

### 资源工厂 (ResourceFactory)

负责创建地图上的资源节点：
- 创建不同类型的资源点（金矿、武器库、农场、补给仓库）
- 在地图上随机分布资源点
- 配置资源点的产出和存储属性

支持的资源节点类型：
- 金矿：提供金币资源
- 武器库：提供武器资源
- 农场：提供食物资源
- 补给仓库：提供辎重资源

## 使用示例

```python
# 创建单位工厂
unit_factory = UnitFactory(world)

# 创建一个平原单位
plains_unit = unit_factory.create_plains_unit(faction_id="red", x=100, y=200)

# 创建建筑工厂
building_factory = BuildingFactory(world)

# 创建一个主基地
headquarters = building_factory.create_headquarters(faction_id="red", x=50, y=50)

# 创建资源工厂
resource_factory = ResourceFactory(world)

# 在地图上创建随机资源点
resource_nodes = resource_factory.create_random_resources(map_data, count=10)
```

## 设计原则

1. **封装复杂性**：工厂隐藏了实体创建和配置的复杂细节
2. **集中配置**：实体的默认属性集中在工厂中定义
3. **灵活创建**：支持自定义参数来创建变体实体
