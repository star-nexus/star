# 战略指挥官 (Strategic Commander)

一个基于Python和Pygame的即时战略游戏，使用实体-组件-系统(ECS)架构开发。

## 游戏概述

《战略指挥官》是一款以军事战略为主题的即时战略游戏，玩家需要通过资源管理、部队调配和战术指挥来击败敌方势力。游戏强调战略思维、态势感知和多单位协同作战。

## 游戏特点

- **多样化地形**：平原、山地、水面、森林、沼泽等不同地形影响单位移动和战斗能力
- **丰富兵种**：包括辎重单位、平原单位、山地单位、水面单位、远程单位和空中单位
- **资源管理**：管理金币、武器、食物和辎重四种基本资源
- **建筑系统**：建造主基地、补给点和战斗工事
- **战术深度**：针对不同地形和敌方单位选择最佳战术
- **阵营机制**：支持多个阵营，每个阵营拥有独特特性

## 技术架构

- **游戏引擎**: 基于自研游戏框架，使用Pygame渲染
- **架构模式**: 实体-组件-系统(ECS)架构
- **地图系统**: 基于格子的地图系统，支持A*寻路算法
- **UI系统**: 模块化UI组件系统

## 模块结构

项目采用模块化设计，各模块职责分明：

- **components/**: 游戏实体的组件定义，包括单位、建筑、资源等
- **systems/**: 游戏系统实现，包括战斗、资源管理、单位控制等
- **factories/**: 实体创建工厂，用于生成各类游戏对象
- **map/**: 地图相关功能，包括地图生成、渲染和寻路
- **ui/**: 用户界面组件，包括资源显示、小地图、信息面板等
- **scenes/**: 游戏场景定义，包括主菜单、游戏主场景等
- **managers/**: 游戏管理器，包括事件、阵营、状态管理等

## 运行游戏

直接运行rts_game.py启动游戏：

```bash
python -m rts.rts_game
```
